# Study Wiki Wall 工程状态总结

> 更新时间：2026-04-27

---

## 1. 工程目标

本工程用于构建一个适合考研题库整理的本地优先 Wiki 学习站，当前重点是：

- 章节 / 知识点 / 题目分层浏览
- 首页统计、单题工作台、错题本、收藏本、导图视图联动
- 本地学习状态持久化
- 为后续 AI / OCR / LLM 导入建立稳定题库结构

## 4. 当前工程边界

当前是：

- 前端原型 + 本地状态原型
- 数据结构验证工程
- 题库导入规则验证工程

当前不是：

- 正式多用户后端系统
- 完整自动 OCR / 文档处理流水线
- 正式部署产品
- 已完成的后台审核平台
